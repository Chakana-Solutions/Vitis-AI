Setting up an Alveo Accelerator Card
=====================================

* DPU DPUCAHX8H/DPUCAHX8H-DWC IP, Alveo™ U50LV card, and Alveo U55C card, are no longer updated since Vitis™ AI 2.5. If you want to use them, refer to `Vitis AI 2.5.0 <https://github.com/Xilinx/Vitis-AI/tree/v2.5>`__.
* DPU DPUCADF8H IP, Alveo U200 card, and Alveo U250 card, are no longer updated since Vitis AI 2.5. If you want to use them, refer to `Vitis AI 2.5.0 <https://github.com/Xilinx/Vitis-AI/tree/v2.5>`__.
* DPU DPUCAHX8L IP, Alveo U50 card, and Alveo U280 card, are no longer updated since Vitis AI 2.0. If you want to use them, refer to `Vitis AI 1.4.1 <https://github.com/Xilinx/Vitis-AI/tree/v1.4.1>`__.
